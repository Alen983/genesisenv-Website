# Installation

Get genesis-env installed and ready to use in your project.

## Quick Install

Install genesis-env using your preferred package manager:

```bash
# npm
npm install -g genesis-env

# yarn
yarn global add genesis-env

# pnpm
pnpm add -g genesis-env
```

## Verify Installation

Check that genesis-env is installed correctly:

```bash
genesis-env --version
```

You should see the version number printed to your terminal.
