# Installation

Get genesis-env installed and ready to use in your project.

## Install as a dependency

From your project root (where `.env.template` or `.env.example` will live):

```bash
npm i genesis-env
```

After a local install, run:

- `npx genesis-env …` from the project root, or  
- `node_modules/.bin/genesis-env …`

## Run without adding to package.json

Uses the registry copy (recommended for trying it out):

```bash
npx genesis-env
```

## Optional global install

```bash
npm i -g genesis-env
```

Then you can run `genesis-env` from your PATH.

## Verify installation

If genesis-env is on your PATH (global install, or via `npx` for a one-off check):

```bash
genesis-env --version
```

You should see the version number printed to your terminal.
